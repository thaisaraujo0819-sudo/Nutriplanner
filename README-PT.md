# NutriPlanner

Sistema web desenvolvido para auxiliar usuários no planejamento alimentar, organização de refeições e geração automática de listas de compras.

🌐 Projeto online
https://nutriplanner-production-0e0d.up.railway.app/nutriplanner

## 📌 Funcionalidades

- Cadastro e login de usuários
- Recuperação de senha via e-mail
- Criação de planos alimentares
- Perfil do usuário
- Geração de lista de compras automática
- Sistema responsivo
- Integração com banco de dados MySQL

## 🚀 Tecnologias utilizadas

- Python
- Flask
- MySQL
- HTML5
- CSS3
- JavaScript
- Jinja2
