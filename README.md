# NutriPlanner

Web application developed to help users plan their diet, organize meals, and automatically generate shopping lists.

🌐 Project online
https://nutriplanner-production-0e0d.up.railway.app/nutriplanner

## 📌 Features

- User registration and login
- Password recovery via email
- Creation of meal plans
- User profile management
- Automatic shopping list generation
- Responsive design
- MySQL database integration

## 🚀 Technologies Used

- Python
- Flask
- MySQL
- HTML5
- CSS3
- JavaScript
- Jinja2
