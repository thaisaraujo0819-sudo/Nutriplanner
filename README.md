# Nutriplanner

Sistema web de planejamento nutricional desenvolvido com Python (Flask).

## Funcionalidades
- Cadastro e login de usuários
- Recuperação de senha com token
- Edição de dados do usuário
- Armazenamento em banco de dados MySQL

## Tecnologias
- Python (Flask)
- HTML, CSS, JavaScript
- MySQL

## Objetivo
Projeto criado para resolver um problema real de organização alimentar.
